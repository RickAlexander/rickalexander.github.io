## Ruby Gem 常用操作

常用的 ruby 命令

<!-- more -->

### 更新Gem自身

```
gem update --system
```

### 更新所有已安装的gem包

```
gem update
```

### 查看本机已安装的所有gem包

```
gem list [--local]
```

### 从Gem源安装gem包

```
gem install [gemname]
```

### 从本机安装gem包

```
gem install -l [gemname].gem
```

### 安装指定版本的gem包

```
gem install [gemname] --version=[ver]
```

### 更新指定的gem包

```
# gem update [gemname]不会升级旧版本的包，此时可以用 gem install [gemname] --version=[ver]

gem update [gemname]
```

### 删除指定的gem包

```
# 此命令将删除所有已安装版本
gem uninstall [gemname]
```

### 删除某指定版本gem

```
gem uninstall [gemname] --version=[ver]
```
